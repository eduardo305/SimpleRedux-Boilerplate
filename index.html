Redux Boilerplate

This boilerplate uses babel and webpack. To use it, simply clone the repo, run npm install followed by the command webpack -d. Finally, just open the index.html in your favorite browser. Branches: - master: has a very simple version using redux-promise middleware - redux-thunk-version: has a version that uses redux-thunk middleware. It demonstrates the power and flexibility of redux-thunk over redux-promise by giving the hability of rendering API errors Please openup your best devtools inspector in order to see the results.
